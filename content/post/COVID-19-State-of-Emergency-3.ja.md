1月8日午前0時をもって1都3県に緊急事態宣言再度発令されました。
今後のイソリア体制について、ご案内いたします。

今回は経済活動を維持しつつ特に飲食店に限定したところもありますが、引き続き最も重要視とされるのは「身体的距離の確保」、「３密（密集、密接、密閉）の回避」ということ、そして、政府から発表された<a href="https://corona.go.jp/prevention/pdf/atarashii_seikatsu.pdf" target="_blank">「新しい生活仕様」</a>及び<a href="https://www.fukushihoken.metro.tokyo.lg.jp/iryo/kansen/kannsenyobouhandbook.files/tominmukehandbook.pdf" target="_blank">「新型コロナウイルス感染症　都民向け感染予防ハンドブック」</a>（令和2年12月17日版）に基づいて、感染拡大予防対策を徹底して行ってだきますようお願いいたします。
イソリアの体制およびオンサイト訪問時の対応については、以下のことを徹底的に守っていただきますようお願いします。

  ※1：顧客の要望に寄り添いつつ感染予防の為、よりリモート対応＆時差訪問での調整に理解を頂いてください。
  ※2：移動手段の柔軟性：公共機関を使用しない選択肢も引き続きご利用いただけます。マイカー通勤の場合、以前からご案内している通り、経費（ガソリン代、高速代、駐車代）の清算をお願いします。満員電車や通勤ラッシュを避けるため、通常の通勤ルートと異なる通勤ルート（より快適で混雑回避可能なルート）は認められますので、事前に管理までご一報をお願いします。

<li>オンサイト訪問においての行動指針</li>
    <ul>
    <li>毎朝体温測定および健康チェック。体調不良時は無理にオンサイト訪問を避けてください。</li>
    <li><font color="red">各顧客先の感染予防対策の指針を遵守してください。また、各々個人のお客様は感染予防対策やCOVID-19に対する温度差をお持ちですので、ITMSの一員として、個人の言動に気を付けてください。イソリアの一員として、会社の感染予防対策に対する姿勢の一貫性を保つ言動を心掛けてください。
  （例：個人のプライベートにおける行動（社会的常識から外れる内容＝外出自粛中の旅行、外出など）を対顧客にお話をしない。）</font></li>
    <li>その他、顧客先で感染予防行動に求められる注意事項について必ず従ってください。それらの注意事項について、個人的に異なる見解をお持ちでも、まずイソリアの一員として顧客の意見に従ってください。</li>
    <li>顧客先での身体的距離の確保：オンサイト作業時、同行エンジニア同士との身体的距離、ＥＵ様との身体距離を保つことに注意しましょう。</li>
    <li>マスクの着用　(予備マスクの持参）。※イソリアの感染防止対策として、ITデスクにマスクの設置可能な顧客に送ります。</li>
    <li>手洗い　（消毒シートや消毒スプレーの持参）。※イソリアの感染防止対策として、ITデスクに消毒液の設置可能な顧客に送ります。</li>
    <li>咳エチケットの徹底。</li>
    <li>顧客先への訪問人数を制限：2名まで。</li>
    <li>作業場所でこまめに換気する（顧客先の責任者に要確認）。</li>
    <li>終日訪問や休憩を伴う訪問時、複数名での同時休憩を避けてください。可能な限り、食事や休憩時には一人ずつ交代しながら休憩しましょう。</li>
    </ul>
<li>日常生活において</li>
    <ul>
    <li>午後8時以降の不要不急の外出の自粛、3密を避ける、買い物や通院など必要な外出も短時間で済ませることなどを要請されていますので、それらの要請に極力従いましょう。</li>
    <li>家族以外の人との会食や飲み会を控えるなど、公私ともに感染対策をよりしっかりと行うようにしてください。</li>
    <li>社員同士の会食も緊急事態宣言期間中は禁止とします。</li>
    </ul>

<li>会社への連絡について</li>
  万が一以下のケースに当てはまる場合、会社として関係各社（保健所、客先など）への連絡義務がありますので、速やかにご連絡いただきますようお願いいたします。
    <ol>
    <li>感染の疑いがある場合</li>
    <li>感染者になってしまった場合</li>
    <li>濃厚接触者になった場合</li>
    <li>ご家族が感染者になってしまった場合</li>
    </ol>
  上記ケースはいずれも該当者には所轄の保健所の指示に従っていただくことが大前提となりますが、社員間の接触および客先での接触などについて、該当者の勤務状況・最終出勤・接触者などの履歴を確認させていただきます。ご協力のほど、よろしくお願いいたします。

  <span style="background-color:yellow">[③濃厚接触者/疑いとなった場合/④同居家族が感染になった場合]</span><br>
  各自治体の保健所の指示に従ってください。ＰＣＲ検査結果が陰性だった場合でも、「患者（確定例）」の感染可能期間の最終曝露日から 14 日間の健康観察が指示されます。<br>
  ご家族が感染者になってしまった場合は、同居者は原則濃厚接触者に該当します。

  ※濃厚接触者の定義<br>
  新型コロナウイルス感染症の患者と感染可能期間（※）に接触した者のうち、次の範囲に該当するものをいいます。
    <ul>
      <li>同居あるいは長時間の接触（車内・航空機内等を含む）</li>
      <li>適切な感染防護なしに新型コロナウイルス感染症患者を診察、看護もしくは介護した</li>
      <li>患者の痰や体液等の汚染物質に直接触れた可能性が高い</li>
      <li>手で触れることのできる距離（<font color="red">目安として1メートル</font>）で、必要な感染予防策なしで、陽性者と<font color="red">15分以上</font>の接触があった者（周辺の環境や接触の状況等個々の状況から患者の感染性を総合的に判断されます）</li>
    </ul>
  ※感染可能期間とは
    <ul> 
      <li>発熱及び咳・呼吸困難などの急性の呼吸器症状を含めた<font color="red">新型コロナウイルス感染症を疑う症状</font>（発熱、咳、呼吸困難、全身倦怠感、咽頭痛、鼻汁・鼻閉、頭痛、関節・筋肉痛、下痢、嘔気・嘔吐など）を呈した<font color="red">2日前から隔離開始まで</font>の間</li>
      <li>無症状病原体保有者の感染可能期間は、<font color="red">陽性確定に係る検体採取日の2日前</font>から入院、自宅や施設等待機開始までの間</li>
    </ul>
<li>感染による休業の取扱いについて</li>
  <ul>
  <li>新型コロナウイルスは指定感染症であり、治癒するまでは就業できません。</li>
  <li>感染経路によっては、労災またはITS傷病給付金などの利用は可能ですので、万が一感染による休業の場合は、管理までご相談ください。</li>
  </ul>
<li>感染者の職場復帰の目安について</li>
  <ul>
  <li>原則として、国が定める基準（PCR 検査による陰性確認）を満たした後に退院となります。</li>
  <li>軽症者で、指定宿泊施設もしくは自宅での療養となった場合も、これに準じた取り扱いとなり、保健所の判断によって療養が終わり、就業制限が解除されます。</li>
  <li>最終的な勤務再開日は、従業員のかかりつけ医か産業医に確認した上で決定します。治療の経過を踏まえて、一定の経過観察期間（在宅勤務や自宅待機等）を設ける場合があります。</li>
  </ul>
  ※（参考） 日本産業衛生学会による「感染した従業員の職場復帰の目安」 
  次の条件をいずれも満たすこと。 
  <ul>
  <li>発症後（ないし診断確定後）に少なくても 10 日が経過している。 </li> 
  <li>解熱後に少なくとも 72 時間が経過しており（a）、発熱以外の症状が改善傾向である（b）。 </li>
  </ul>
  （a）解熱剤を含む症状を緩和させる薬剤を服用していない <br>
  （b）咳・倦怠感・呼吸苦などの症状（ただし味覚・嗅覚障害については遷延することがある）<br>
